# RUCK2020
R User Conference in Korea 2020

## 컨퍼런스의 비전

- 참여와 발굴

참여는 운영위의 운영 방식을, 발굴은 운영위의 목표를 의미합니다.

최초로 온라인으로 진행되는 만큼 국내는 물론 기존에 참여가 어려웠던 여러 방면과 지역의 분들을 모시고 참여를 기대할 수 있는 컨퍼런스가 되고자 합니다.

참여를 독려하기 위해서 최대한 공개하는 운영위로 운영하겠습니다.
